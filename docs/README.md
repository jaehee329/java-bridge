# 미션 - 다리 건너기

## 구현 기능 목록

- 입력 처리 기능
  - 입력 기능
    - [ ] 다리의 길이를 입력받는 기능
    - [ ] 사용자가 이동할 칸을 입력받는 기능
    - [ ] 재시도 여부를 입력받는 기능
  - 입력 검증 기능
    - [X] 다리의 길이가 3~20사이인지 검증하는 기능
    - [ ] 이동할 칸이 U또는 D인지 검증하는 기능
    - [ ] 재시도 여부가 R또는 Q인지 검증하는 기능

- 출력 처리 기능
  - [ ] 현재까지의 이동 상태를 출력하는 기능
  - [ ] 게임의 최종 결과를 출력하는 기능
  - [ ] 게임의 성공 여부를 출력하는 기능
